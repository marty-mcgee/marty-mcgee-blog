## 🚀 Getting Started

### 📦 Dependencies

- next 14.1+
- node v20.10+
- npm v10.2+
- tailwind v3.3+

### 👉 Install Dependencies

```bash
yarn install
```

### 👉 Development Command

```bash
yarn dev
```

### 👉 Build Command

```bash
yarn build
```

### 👉 Build and Run With Docker

```bash
docker build -t marty-mcgee-blog .
docker run -p 3000:3000 marty-mcgee-blog
```
